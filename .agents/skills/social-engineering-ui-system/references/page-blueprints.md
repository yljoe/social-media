# Page Blueprints

## 1. Dashboard

### 結構

- 頁面標題
- KPI 卡片列
- 主列表或主表格
- 補充區塊

### 適用頁面

- 總覽
- 最近專案
- 成本概況
- 系統狀態

### 禁止

- KPI 太多
- 補充卡片比主列表搶眼

## 2. Generation Page

### 結構

- 頁面標題與簡述
- Stepper 或模式切換
- 左側設定面板
- 右側預覽面板
- 主 CTA

### 適用頁面

- 郵件生成
- Landing page 生成
- Mobile mockup 生成
- 影片場景設定

### 必備

- 設定與預覽同頁
- 生成結果即時可見
- 預覽 tabs 清楚

## 3. Management Page

### 結構

- 頁面標題
- 工具列：搜尋、篩選、建立
- 主表格
- 分頁或統計摘要
- 編輯器可位於側邊、下方或獨立區塊

### 適用頁面

- Provider 管理
- 素材管理
- 檔案管理
- 專案管理

### 必備欄位

- 名稱
- 類型
- 狀態
- 更新時間
- 動作

## 4. Logs / Costs Page

### 結構

- 摘要區
- 記錄表格
- 詳細內容區

### 適用頁面

- API logs
- 成本追蹤
- 任務執行歷史

### 必備欄位

- 時間
- 事件名稱
- 狀態
- Provider
- 成本或耗時
- 詳情入口
